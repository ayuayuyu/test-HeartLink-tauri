# Tauri + React

### Reactルーティングのインストール
```bash
npm i -S react-router-dom
```
### destrのインストール
```bash
npm i destr
```
### MUIのインストール
```bash
npm install @mui/material @emotion/react @emotion/styled
```
### chart.jsのインストール
```bash
npm i chart.js
npm i react-chartjs-2
```
### WebSocketのインストール
```bash
npm install --save reconnecting-websocket
```
